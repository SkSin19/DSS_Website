# Digital Security Solutions — Website

A modern product catalogue and enquiry website for **Digital Security Solutions (DSS)** — a CCTV & security products business.

---

## Tech Stack

- **TypeScript** — Primary language across the entire codebase
- **React** — UI component library
- **Next.js** — Full-stack React framework (App Router, SSR)
- **Tailwind CSS** — Utility-first styling
- **GSAP** — Animations & scroll effects
- **Three.js** — 3D rendering
- **Axios** — API communication
- **Node.js** — Runtime environment

---

## Getting Started

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) in your browser.
